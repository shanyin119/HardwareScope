# 安全政策

## 支持的版本

当前仅维护最新公开版本。安全修复会发布在新的补丁版本中。

## 报告安全问题

请优先使用本仓库 Security 页面中的“Report a vulnerability”私密报告功能。不要在公开 Issue 中发布漏洞利用代码、个人信息或尚未修复的详细攻击步骤。

如果私密报告入口暂不可用，可以新建一个不包含敏感细节的公开 Issue，请求维护者开启私密沟通渠道。

报告时请尽量包含：

- 受影响版本和 Windows 版本；
- 问题影响及重现所需的最小步骤；
- 已采取的缓解措施；
- 是否已经向其他人公开。

维护者会确认收到报告、评估影响，并在修复可用后协调披露。
