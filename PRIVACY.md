# 隐私政策

最后更新：2026-07-22

## 摘要

别离检测工具在本机读取硬件与传感器数据，不会将硬件信息、温度、应用设置或导出内容传输到开发者或任何分析服务。本项目不包含遥测、广告、用户账户或后台数据收集服务。

## 本地读取与保存

- 应用通过 Windows 系统接口和 LibreHardwareMonitor 读取硬件名称、规格、标识和传感器读数。
- 设置保存在 `%LocalAppData%\别离检测工具\settings.json`。
- 用户启用“开机启动”时，应用会在当前用户的 Windows 启动项中保存程序路径；关闭该选项时会移除此项。
- 用户主动导出时，选中的硬件信息只会写入用户指定的 TXT、DOCX 或 XLSX 文件。

## 网络访问

应用没有遥测或自动更新功能。唯一的应用内下载行为是：用户明确确认安装处理器/主板温度支持后，从以下 PawnIO 官方发布地址下载 PawnIO 2.2 安装程序：

`https://github.com/namazso/PawnIO.Setup/releases/download/2.2.0/PawnIO_setup.exe`

应用会在启动安装程序前核对固定的 SHA-256。该请求会受到 GitHub 自身隐私政策约束。应用不会在请求中附加硬件信息或应用设置。

## 管理员权限和驱动

应用请求管理员权限，以读取受保护的低层硬件信息。PawnIO 是单独安装和卸载的第三方开源组件，不包含在本项目安装包中。卸载别离检测工具不会自动卸载 PawnIO，也不会自动删除本地设置；用户可以通过 Windows“已安装的应用”卸载 PawnIO，并手动删除上述设置目录。

## 第三方组件

应用使用的开源库及构建工具列在 [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md)。使用者可能受相应项目的许可证或隐私政策约束。

## 变更

如果未来加入任何联网、遥测或数据传输功能，将在实现前更新本政策，并在应用中提供明确说明和可关闭的选择。
